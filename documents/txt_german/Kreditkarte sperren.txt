Wenn es kein Notfall ist und Sie einfach einzelne
Karte sperren:
 
 
Innerhalb von 3 Werktagen erhalten Sie eine
neue Karte mit neuer Nummer. Die PIN Ihrer
Karte können Sie immer
via "Kartenfunktionen" → "Ihr Code
PIN" abrufen.
Wählen Sie "Ihre Produkte” ganz oben im
Menü.
Gehen Sie zu "Karten".
Wählen Sie die Karte, die Sie sperren
möchten.
Sie können die entsprechende Karte auch
direkt in der George Übersicht auswählen.
Gehen Sie zu den "Kartenfunktionen".
Scrollen Sie zu "Service/Kartenstatus".
Klicken Sie auf "Sperren".
Wählen Sie einen Grund für die
Kartensperre.
Beenden Sie den Vorgang und sperren Sie
Ihre Karte mit "Jetzt sperren".
Ihre Karte ist nun umgehend gesperrt.

Einsatzbereiche Ihrer Karte deaktivieren
möchten, z. B. die Möglichkeit zur
Bargeldbehebung, ist das auch via George
möglich.
 
Wenn Ihre Karte kaputt ist oder Sie
beispielsweise kurz vor dem Ablaufdatum Ihrer
Karte verreisen sollten, können Sie bereits
vorher eine neue Karte nachbestellen. Ihre alte
Karte wird dann automatisch gesperrt.
